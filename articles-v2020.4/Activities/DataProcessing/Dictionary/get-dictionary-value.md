# 获取值

## 视频示例

## 概述

获取字典中 key 对应的 value 的值。

## 属性

### 基本

参见 [通用配置项](../../Appendix/CommonConfigurationItems.md)。

### 输入

- **字典**：需要获取键值所在的字典。
- **键**：字典中的键。

### 输出

- **值**：字典中的值。

## 使用示例

**前置必要组件**：[初始化字典](../Dictionary/InitializeDictionaryActivity.md)、 [添加键值对](../Dictionary/AddDictionaryActivity.md)

**此流程执行逻辑**：获取字典 `D` 中对应的键 `"key1"` 的值。
