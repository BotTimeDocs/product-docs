# CV 输入文本

## 视频示例

## 概述

向用户界面元素发送需要输入的文本内容。

> **说明：**
>
> 此组件需在“指定 CV 窗口”组件中使用。

## 属性

### 基本

参见 [通用配置项](./../Appendix/CommonConfigurationItems.md)。

### 输入

- **文本**：需要输入的文本内容。

### 目标

- **描述符**：自动生成的用于定位识别的元素信息。

## 使用示例

**此流程执行逻辑**：在指定的用户界面中的指定输入框中，自动输入 `cvc` 文本内容。
