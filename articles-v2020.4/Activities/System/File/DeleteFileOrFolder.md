# 删除文件/文件夹

删除指定文件或文件夹

## 属性

输入

- **路径** ：删除文件或文件夹的路径。支持相对和绝对路径。可在组件面板点击弹出对话框，选择要删除的文件或文件夹；亦支持手动输入路径，若路径不存在，默认成功。仅支持字符串变量和字符串
- **删除类型** ：下拉框选择删除文件或文件夹

## 操作样例
1. 拖入**删除文件/文件夹**组件到设计面板，双击进入组件内部。

2. 输入删除文件/文件夹的全路径，支持相对和绝对路径。
