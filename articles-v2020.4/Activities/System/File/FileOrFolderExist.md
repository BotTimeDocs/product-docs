# 文件/文件夹是否存在

判断指定文件或文件夹是否存在，并返回结果

## 属性

### 输入

- **类型** ：下拉框选择判断类型为文件或文件夹
- **路径** ：执行判断的文件或文件夹路径。支持相对和绝对路径。仅支持字符串变量和字符串

### 输出

- **路径是否存在** ：将判断结果存储在此变量
