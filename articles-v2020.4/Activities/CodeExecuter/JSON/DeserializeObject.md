# 反序列化

## 视频示例

## 概述

将JSON字符串反序列化为.NET对象。

## 属性

### 基本

参见 [通用配置项](../../Appendix/CommonConfigurationItems.md)。

### 输入

- **JSON字符串** ：输入欲被反序列化为JSON数据。

### 输出

- **结果**：输入被反序列化后的对象；仅支持变量。

## 使用示例

**此流程执行逻辑**：将定义的变量`json字符串`反序列化为`json对象`输出。
