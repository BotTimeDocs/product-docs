# 云扩RPA平台

**云扩RPA平台**（Encoo Automate）是云扩科技自主开发的企业级RPA（机器人流程自动化）平台，旨在为企业客户提供简单易用、安全稳定的智能自动化解决方案。云扩RPA平台一共有三个核心组件，分别是云扩RPA编辑器(Encoo Studio)、云扩RPA机器人(Encoo Robot)、云扩RPA控制台（Encoo Console），三个组件相互配合，帮助企业快速实现业务流程的自动化。
